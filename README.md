# Бот для заказа такси "Тройки"

## Используемые технологии
- VK API
- TMApi
- GenApi (нейросеть)

## Планы
Клиент общается с ботом как с живым человеком, главная фишка, что нет кнопок, живой диалог
Можно заказать такси, узнать цену, записаться на собеседование для работы в компании
Планируется создать админ панель, через которую нейросеть будет генерировать посты в группу и администрировать данную группу

Подразумевается сделать сервер с API, который можно будет использовать для сайта или других сервисов
В будущем создать собственную экосистему такси, то есть без использования BitMaster
